# Minecraft Mod Database Management
An application to manage Minecraft mods. Create for the sole purpose of experimenting with SQL database and how to inplement it with Unity.  
To use the application: download and unzip the file "Minecraft Mod Database Searcher (Build).zip" and open the application in the unzipped folder (not the UnityCrashHandler64).  
To open the project: download the whole folder and open it in Unity (version 6000.0.46+).
